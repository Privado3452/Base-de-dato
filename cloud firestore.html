Clientes

Cloud Firestore es una base de datos NoSQL de documentos desarrollada por Google, que ofrece una capa de base de datos en tiempo real para aplicaciones web y móviles. Se basa en la tecnología de Firebase, una plataforma de desarrollo de aplicaciones móviles de Google.




Caracteristicas

  • Base de datos en tiempo real: Las actualizaciones de datos se reflejan instantáneamente en las aplicaciones conectadas.

  • Escalabilidad sin servidor: Se escala automáticamente según la demanda, sin necesidad de administrar infraestructura.

  • Integración con Firebase: Se integra a la perfección con otros servicios de Firebase, como autenticación y almacenamiento en la nube.

  • Seguridad robusta: Ofrece medidas de seguridad integradas para proteger los datos.




Funcionamiento

Cloud Firestore almacena datos en documentos JSON, organizados en colecciones. Las actualizaciones de datos se propagan en tiempo real a las aplicaciones conectadas mediante WebSockets.




Desventaja

  • Costo: Puede ser más costoso que otras bases de datos NoSQL, especialmente para grandes volúmenes de datos.

  • Dependencia de Google: Requiere una cuenta de Google Cloud Platform y está sujeto a las políticas de Google.

  • Menos flexibilidad de consultas: Las consultas no son tan potentes como en otras bases de datos NoSQL.

Página oficial de documentación: https://firebase.google.com/docs/firestore
